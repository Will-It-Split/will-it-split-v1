# 🪓 Will It Split?

A satisfying log-chopping incremental game where you upgrade axes, unlock new log types, and break tougher wood with progression, strategy, and upgrades.

---

## 🎮 Features (V1)
- 🪓 26 upgradeable axes with strengths/weaknesses  
- 🪵 16 log types (wood → exotic → magical tiers)  
- 👹 Boss logs with big rewards  
- ⭐ Prestige system  
- 🏆 Achievements system  
- 📈 Progression-based gameplay loop  

---

## 🔥 Gameplay
Click/tap to chop logs, earn coins, upgrade axes, and unlock stronger logs as you progress.

---

## 🕹️ How to Play
1. Click or tap the log to chop it 🪓  
2. Reduce its HP until it splits  
3. Earn coins from each split 💰  
4. Buy stronger axes in the shop  
5. Unlock new log types as you progress 🪵  
6. Defeat boss logs for big rewards 👹  
7. Use prestige to reset and become stronger ⭐
